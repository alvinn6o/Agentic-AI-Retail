# Markdown / Discount Policy SOP

## Trigger Conditions
- Items with zero sales for 60+ days should be considered for markdown
- Items with high return rate (>15%) should be investigated before markdown

## Discount Tiers
| Weeks Without Sale | Discount |
|--------------------|----------|
| 4–8 weeks          | 10%      |
| 8–12 weeks         | 25%      |
| 12–16 weeks        | 40%      |
| 16+ weeks          | 50%+ or clearance |

## Approval
- Up to 25% discount: Category manager can approve
- 25%–50% discount: Operations manager approval required
- Over 50% or clearance: Director approval required

## Exclusions
- Seasonal items: apply seasonal discount only during off-season
- High-value items (unit price > £50): escalate to director before any markdown
